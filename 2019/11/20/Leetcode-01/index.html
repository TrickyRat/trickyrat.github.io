两数之和（Two Sum）

描述

Given an array of integers, return indices of the two numbers such that they add up to a specific target.
给定一个整数数组 nums 和一个目标值target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。

You may assume that each input would have exactly one solution, and you may not use the same element twice.
你可以假设每种输入只会对应一个答案。但是，你不能重复利用这个数组中同样的元素。

示例

Given nums = [2, 7, 11, 15], target = 9,
Because nums[0] + nums[1] = 2 + 7 = 9, return [0, 1].

解题

思路

一开始想到就是使用暴力的方式直接循环两次，检测数组中是否有差值的元素。每个元素都需要遍历一次数组,每个元素都会耗时O(n),所以时间复杂度为O(n^2),空间复杂度O(1)。

代码

class Solution
{
    public int[] TwoSum(int[] nums, int target)
    {
        int len = nums.Length;
        for (int i = 0; i < len; i++)
        {
            for (int j = 0; j < len; j++)
            {
                if (nums[j] = target - nums[i])
                    return new int[] { i, j };
            }
        }
        return new int[];
    }
}

看到求两个数的和，那意思就是看看给定数组nums中，是否有一对值和为target的组合；这样其实就能很快速想到键值对的存在，所以可以考虑使用hashmap来尝试解题。将数组的元素x和target-x作为键值对，遍历整个数组，一边添加一边查询hashmap是否存在当前元素的目标值，如果存在，则将组合以数组形式返回，相反则没有。
时间复杂度只遍历一次数组，hashmap查询消耗O(1)时间，则总的时间复杂度为O(n)，
空间复杂度，因为使用了一个hashmap来存储键值对，需要额外的空间，最坏情况下需要消耗O(n)，所以空间复杂度为O(n)。
